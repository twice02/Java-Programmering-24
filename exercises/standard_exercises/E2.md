**2.1**  
> What is the value of **`y`** after the following conditional statements?  

a.
```java
int x = 3, y = 8;

if (x < y) {
    y = x + y;
} else {
    y = y - x;
}
```
b.
```java
int x = 12, y = 200;

if (x == y) {
    y = 80;
} else {
    y = x;
}
```
c.
```java
int x = 12, y = 0;
y = x * x - 2 * x + 30;
if (x < y) {
    y = 1;
} else {
    y = 0;
}
```

d. `Why is the code in 2.1c not a good way to write operations? How could it be improved?`

**2.2**
What is the value of `result` after the following loop statements? Explain why.  

a.
```java
int result = 0;
for (int i = 1; i < 5; i++)
    result = result + i;
```
b. 
```java
int result = 140;
for (int i = 20; i > 15; i--)
    result = result - i;
```
c. 
```java
int result = 0;
for (int x = 1, y = 10; x < 5; x++, y--)
    result = result + x * y;
```
d. 
```java
int result = 0;
while (result <= 100)
    result = result + 16;
```
e. 
```java
int result = 0, a = 0;
while (result <= 100 & a <= 10) {
    result = result + a;

}
```
